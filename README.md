# react-native-speech-recognization

## Getting started

`yarn add react-native-speech-recognization`

Or

`$ npm install react-native-speech-recognization --save`

* ios

rect-native >= 0.60

You can run:
```
$ npx pod install
```

Or

```
cd ios && pod install
```

* android



## Usage
```javascript
import {DictationPanel, RNDictation} from 'react-native-speech-recognization';
```
